# USB 2.1

USB 2.1 is treated as a USB 2.0 baseline plus applicable ECNs/addenda, especially LPM-related updates, unless an independent official base specification is identified.

This legacy page is not the canonical visible LLM wiki surface.
Use `specs/version_source_map.md` and `specs/verification_status.md` for current repo-local claim boundaries.

## Claim Boundary

- Do not describe USB 2.1 as an independent base hub model without official standalone evidence.
- USB 2.1 entries must link back to USB 2.0 baseline context.
- This page does not claim USB 2.1 hub behavior is verified.
- This page must not be used to override confirmed project facts in consuming repositories.
