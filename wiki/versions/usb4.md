# USB4

USB4 is handled as topology/router context in this repository.

## Claim Boundary

- Do not assume USB4 behavior is equivalent to USB 2.0 or USB 3.x hub class behavior.
- Any equivalence claim requires section-level evidence from normative USB4 sources.
- Use USB4 pages to define boundary context first, then map deltas carefully.
