# Hub Descriptor

This page explains hub descriptor concepts and cross-version boundaries.

## USB 2.0 Baseline

Use USB 2.0 as the baseline reference for classic hub descriptor semantics.

## USB 3.2 Delta

Track SuperSpeed-related hub descriptor deltas separately from USB 2.0 baseline.

## USB4 Boundary

Do not project USB 2.0 or USB 3.x descriptor semantics into USB4 without explicit normative evidence.
