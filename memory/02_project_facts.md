# Project Facts

This is a spec reference repository, not a firmware implementation repo.

## Authoritative Source

- USB 2.0 Specification, Revision 2.0 (April 27, 2000), Chapters 11 and related
- USB Hub Class Specification

## Scope

- Hub class descriptor definitions: spec section 11.23
- Hub class request definitions: spec section 11.24.2
- Port status and change bits: spec section 11.24.2.7
- Transaction Translator rules: spec section 11.17–11.18
